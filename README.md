# JavaScript Bits
JavaScript 语言常用的功能片段，可能是原生方法的补充，可能是一些常用UI组件，React组件的制作。
这个片段库更多的是为了方便后续浏览，所以并没有计划和 `lodash` 等函数库一样

## Install
安装依赖，为搭建一个浏览器环境，使用了一个简单的 `server`
```bash
npm install
npm install -g http-server
npm run dev
```
